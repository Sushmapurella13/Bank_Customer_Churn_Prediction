# Bank_Customer_Churn_Prediction
## Project Overview
This project predicts whether a bank customer will leave the bank (churn) or stay, using machine learning techniques. By analyzing customer information such as credit score, age, balance, and account activity, the model helps identify customers who are likely to churn.

## Technologies Used
- Python
- Machine Learning
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Dataset
The dataset contains customer information including:
- Customer ID
- Credit Score
- Country
- Gender
- Age
- Tenure
- Balance
- Number of Products
- Credit Card Status
- Active Member Status
- Estimated Salary
- Churn (Target Variable)

## Methodology
1. Data preprocessing and cleaning  
2. Feature selection and encoding  
3. Training machine learning models  
4. Model evaluation using accuracy and confusion matrix

## Model Used
- Logistic Regression
- Perceptron
- KNN

## Result
The trained model predicts whether a customer will churn based on their banking behavior and demographic information.
